# 🏛️ The Celes Archive Project

> **Mission:** Preserving open-source knowledge by archiving documentation in the lightweight, portable, and versatile `.celes` format.

## ❓ What is this?
Open-source projects often disappear or change their documentation URLs, leading to "link rot." This account is dedicated to forking vital open-source projects, stripping away the source code, and transforming their documentation into **.celes** files—a format designed for longevity and easy distribution.

## 🛠️ Why .celes?
The `.celes` format is built for accessibility across multiple platforms. We currently support three primary ways to interact with archived data:
1. **The Human Way:** Readable directly as a structured document.
2. **The Desktop Way:** A Python-based application (available as an EXE) for distribution and scripting.
3. **The Web Way:** A dedicated browser extension for seamless viewing.

## 🚀 How We Archive
1. **Fork:** We fork active or at-risk open-source repositories.
2. **Prune:** We remove the codebase to keep the archive lightweight and focused solely on knowledge.
3. **Convert:** Documentation is structured into `.celes` files.
4. **Distribute:** The final archive is hosted here for the community.
